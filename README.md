# Mix Comps — Artist Offer Landing Page

Landing page for artists, producers, and rights holders to submit multitracks/stems to [MixComp](https://mixcomps.com/) and turn their song into a mixing competition.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Page sections

1. **Hero** — Core offer and CTAs
2. **What artists get** — Seven key benefits
3. **How it works** — Six-step process
4. **Example payout** — Compensation tiers + example campaign
5. **Rights & safety** — Rights protection and Safari Audio trust
6. **FAQ** — Common artist questions
7. **Application form** — Full submission form per the artist offer plan

Design matches [mixcomps.com](https://mixcomps.com/) — cream background, orange accents, bold uppercase typography, pill buttons, and black borders.
